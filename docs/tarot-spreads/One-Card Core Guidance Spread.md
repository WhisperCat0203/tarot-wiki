# One-Card Core Guidance Spread

- *Suitable for*: Yes/No questions, overall energy, direct guidance
- *Example questions*:  
  What is my overall energy today?  
  Does this situation show signs of completion?  
  Can this project succeed?

- *Card position meaning*:  
  1: Core message of the present moment / Guidance for you / The essence of the situation
